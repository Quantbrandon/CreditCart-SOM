# CreditCart-SOM
Application of Self-Organizing Maps (SOM) to learn and visualize the topological structure of a credit card customer dataset. This project explores customer patterns and relationships using unsupervised neural networks.

The purpose of this project is to create a step-by-step animation that illustrates how a Self-Organizing Map (SOM) learns the underlying topology of a dataset.

Through this visualization, we aim to provide an intuitive understanding of how the SOM algorithm organizes high-dimensional data by preserving its structure in a lower-dimensional representation, showing the evolution of the map during the training process.

## Data Source
This project uses the Credit Card Dataset from Kaggle:
https://www.kaggle.com/datasets/arjunbhasin2013/ccdata
